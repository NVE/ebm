# tiks26



# Beskrivelse

TIKS26-datasettet inneholder oppdaterte forutsetninger for framskriving av energibruk i bygg. De viktigste endringene gjelder riving, innfasing av LED, virkningsgrader for oppvarmingsløsninger, boligstørrelser, fordeling av nye boliger og kalibrering mot oppdatert statistikk. Datasettet brukes som oppdatert grunnlag for framskriving av energibruk i bygg i TIKS26.

Datasettet gir et lavere utgangspunkt for framskriving av kraftforbruk i bygg enn tidligere kalibrering, samtidig som enkelte endringer, særlig knyttet til riving og oppvarmingsløsninger, trekker energibruken opp isolert sett.
